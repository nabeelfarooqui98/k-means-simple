# k-means-simple
A simple k means clustering solver


## Goal

* Put data points in text file
* Put initial centroids in text file
* Run program

## Input file format

N followed by N data points
M followed by M centroids

example:

7

1 1

1.5 2

3 4

5 7

3.5 5

4.5 5

3.5 4.5

3

1 1

5 7

3.5 4.5
